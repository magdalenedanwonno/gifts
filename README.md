 Birthday GIFt Site: Ova's 26th Birthday

This project is a playful, interactive birthday tribute site created for a friend named Ova. It features a "gift-opening" experience where users can hover over placeholder images to reveal various celebratory GIFs.

🚀 Features

* Interactive GIF Sections: Multiple sections that reveal specific GIFs upon hovering, simulating the opening of a gift.
* Custom Typography: Integrated with Google Fonts using the 'Happy Monkey' cursive font for a festive feel.
* Dynamic Visuals: 
    . A vibrant linear-gradient background (`#EE6983` to `#FFC4C4`).
    . Circular profile image styling for the "BFF" photo.
    . Text shadows applied across all headings for improved readability against the colorful background.
* Responsive Layout: Centered content using CSS Flexbox for the header and automated margins for the gift sections.

🛠️ Technical Breakdown

Core Technologies
* HTML5: Semantic structure for the birthday message and gift sections.
* CSS3: Handles the gradients, hover states, and layout.
* Giphy : High-quality celebratory GIFs are pulled from Giphy to populate the gift sections.

File Structure
* `index.html`: The main markup containing the birthday message and gift-section containers.
* `styles.css`: The comprehensive styling rules, including hover effects and the color palette.
* `mary.jpg`: The profile image used for the birthday person.

🎨 Interactive Elements

The site uses specific IDs to trigger different reactions:
* `#gift-img-happy`: Shows a "happy" reaction GIF.
* `#gift-img-hot`: Shows a reaction to someone entering a room.
* `#gift-img-genius`: Displays a "genius" description GIF.
* `#gift-img-badass`: Displays a "badass" person GIF.
* `#gift-img-cheers`: A final celebratory cheers GIF.

🎓 Background
This project was developed as part of a Scrimba HTML & CSS course to practice DOM structure, external asset integration, and interactive CSS transitions.
